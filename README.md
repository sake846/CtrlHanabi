# CtrlHanabi

Ctrlキーを2回連打すると、マウスカーソル位置で花火が開くWindows常駐アプリです。

## 実装済み（Phase 3まで）
- Ctrlダブルタップ検知（閾値設定あり）
- マウス位置の花火オーバーレイ表示
- タスクトレイ常駐
- Windows起動時の自動起動ON/OFF
- 設定ファイル（LocalAppData）読み書き
- 打ち上げ軌道つき演出（ロケット上昇→爆発）
- 花火種類: 菊 / 牡丹 / 冠菊（かむろぎく）
- 全種類で3D球状に開く爆発表現
- 冠菊は橙〜金色系パレット、流れ星風の細長い尾、中心優先＋全体ソフトフェード
- 色相ランダム、残像、発光エフェクト
- ディスプレイ構成変更時の仮想スクリーン追従

## 補足
- ターゲット: `.NET 10` + `WPF`
- 設定ファイル: `%LocalAppData%/CtrlHanabi/settings.json`
